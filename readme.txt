== Description ==



Embed flickspire movies in your wordpress blog.


Currently requires an active flickspire viral video system account to attain movie codes.



== Installation ==



1. Upload 'flickspire' to the '/wp-content/plugins/' directory

1. Activate the plugin through the 'Plugins' menu in WordPress

1. Use the [fsmovie]movie=MovieCode&custcode=YourFlickspireCustomerCode[/fsmovie] shortcode in your posts



== Frequently asked ques tions ==



= Can anyone use this plugin? =

No, this plugin currently requires an active flickspire viral video system account to attain movie codes.



= How do I embed my movies? =
You can embed your flickspire movies using this shortcode syntax:

[fsmovie]movie=MovieCode&custcode=YourFlickspireCustomerCode[/fsmovie]
